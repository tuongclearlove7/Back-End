## Git commands

```js

- Tạo nhánh

=>  git checkout -b cl7

- Kiểm tra các khánh 

=> git branch

- Đẩy code lên nhánh

=> git add .
=> git commit -m "updated"
=> git push origin cl7

- Merge code trên giao diện github

- Resolve conflicts code 

=> git pull 
=> Rồi vào file vừa pull về sửa lại nội dung muốn push

=> Sau đó push lên lại github


```
